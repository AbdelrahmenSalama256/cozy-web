# Cozy Web

Responsive multi-page e-commerce website built with HTML, CSS, and JavaScript.

## Pages
- Home
- Category listing
- Product details
- Shopping cart
- Checkout
- User profile
- Orders and receipts
- Contact
- Wishlist

## Features
- Responsive design
- Modern UI/UX
- Product catalog
- Cart management
- User authentication pages

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Getting Started

Open `index.html` in your browser or use a local server:

```bash
# Using Python
python -m http.server 8000
# Then visit http://localhost:8000
```
